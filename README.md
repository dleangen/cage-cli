# cage-cli

`cage-cli` is the command-line tool for scaffolding and managing CAGE-enabled projects. It seeds the canonical `agents.md` protocol into new projects and helps maintain CAGE conventions over time.

> **Status:** This package is under active development. CLI commands are not yet available.

## Installation

```bash
npm install @dleangen/cage-cli
```

## CAGE ecosystem

| Package | Description |
|---------|-------------|
| [cage-core](https://github.com/dleangen/cage-core) | Agent protocol specification |
| **cage-cli** | CLI for scaffolding CAGE-enabled projects (this package) |
| [cage-history](https://github.com/dleangen/cage-history) | Conversation history conventions |
| [cage-management](https://github.com/dleangen/cage-management) | Management tooling for CAGE projects |
